void infixToPrefix ( char  infix[100],
char  prefix[100] 
)

This function converts infix expression to prefix expression.

Here, in this function, we first reverse the infix expression and apply the conversion algorithm to get the prefix expression according to the precedence rules by using a user-defined stack.

Attention
It is important to note that the input expression is well-parenthesised in Backus-Naur Form(BNF).
